## Overview
This project explores the Netflix Titles dataset to better understand the platform's content library. The analysis focuses on identifying trends in movies and TV shows, content distribution across countries, popular directors, ratings, genres, and release patterns using Python and data visualization techniques.
The goal of this project was to practice data cleaning, exploratory data analysis (EDA), and visualization while extracting meaningful insights from real-world data.

## Features
- Data cleaning and preprocessing
- Date formatting and feature extraction
- Exploratory Data Analysis (EDA)
- Visualizations using Matplotlib and Seaborn
- Analysis of movies vs TV shows
- Country-wise content distribution
- Director and cast analysis
- Content ratings and release year trends
- Genre distribution and content insights
